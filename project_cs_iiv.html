Python Client Registration Automation

About a year in to my journey of learning to code I became increasingly aware that one of my main administration tasks in my role at NCVO could be automated.

The workflow for processing client registrations for NCVO’s Investing in Volunteers quality standard was very admin-heavy, involving the transfer of client registration data between systems, spreadsheets, contracts & templates. Each client registration would take around an hour to process.

The solution

I chose Python as the language to learn & write the automation script in, due to the large amount of documentation available dedicated to working with Microsoft Office packages used in the workflow.

The workflow

I split the current admin process down into 50 individual steps, to create a workflow that I could automate. View a copy of this workflow.

Creating the script

The script drew heavily on Python’s docx and openpyxl modules to interact with Microsoft Word & Excel documents.

The functions in these modules allowed me to break the Word & Excel sheets down into separate objects that can be accessed by the script.

I created custom modules for each broad stage of the workflow, and started automating the workflow step by step, moving the data submitted in a client's registation form into our monitoring documents, contracts and templates.

A number of client-submitted variables needed checking & inserting manually into the script before it could run, so I set these up as user inputs prompted by the Python IDLE shell at the start of the script.

View the final script.

End result

I was delighted with the end result - the processing time for each registration had been cut down to around 5 minutes, including the user input at the beginning of the script.

Creating the automation required me to learn & research a range of programming techniques including regexes, input validation, use of the command line, automated web browser searching, and plenty of hours on Stack Overflow!

Being able use my coding learning & experience to solve a real-world problem was eye-opening for me, and definitely helped to push me closer to my decision to pursue a career in the field.
